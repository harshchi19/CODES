## üöÄ Getting Started

### Prerequisites
```bash
# Python 3.x
python --version

# Install required Python packages
pip install numpy pandas scikit-learn matplotlib seaborn jupyter
```

### Running Jupyter Notebooks
```bash
# Navigate to the repository
cd CODES

# Start Jupyter Notebook
jupyter notebook

# Open any .ipynb file from the browser interface
```

### Running Python Scripts
```bash
# Example: Running DVR algorithm
python "ML for Satyavrat/DVR.py"

# Example: Running TCP Server
python "ML for Satyavrat/TCP_Server.py"
```

### Running Java Programs
```bash
# Compile
javac "ML for Satyavrat/TCP_Client.java"

# Run
java TCP_Client
```

### Running C Programs
```bash
# Compile
gcc "ML for Satyavrat/DVR.c" -o dvr

# Run
./dvr
```
